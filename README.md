### `块文本文件说明书`
以下是块文本文件（Block-Text-File文件，简称BTF文件）的格式定义说明书，`欢迎转载，但请勿改动以免发生误解`

#### 文件编码
* 文件编码，固定为UTF-8

#### 块名及其的文本
* 块名：使用中括号包围，在一行的开始处，同一行中括号以后的文字忽视
* 块名对应的文本：从块名的下一行开始，到该块结束为止，中间的内容就是
* 块结束标志：下一块名开始之前；
